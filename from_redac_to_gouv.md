# FROM REDAC TO GOUV

## Nature du fichier
Handoff court de REDAC vers GOUV.
Ce fichier est remplaçable à chaque cycle.
Il ne sert pas d’archive longue.

## Format attendu
- build source : `rXX`
- travail réel effectué
- limites rencontrées
- risques observés
- besoin éventuel de recadrage
- usage ou non de `faculta_redacteur.md`

## Gabarit minimal à réécrire à chaque cycle

- build_source: rXX
- build_travaillee: rXX
- lot_execute: décrire en 1 à 3 lignes ce qui a réellement été fait
- fichiers_touches: liste courte des zones réellement modifiées
- resultat: ce qui est meilleur maintenant
- non_fait: ce qui est resté intact ou volontairement reporté
- risques: incohérences, dérives, trous, gouvernance périmée, dette de structure
- faculta_redacteur_utilise: oui/non
- commentaire_redac_a_signaler: vide si rien d’important
- demande_a_gouv: recadrage / validation / nouveau lot / nettoyage / évaluation

## Règles
- rester concret ;
- ne pas écrire un journal intime ;
- ne pas faire de théorie générale ;
- ne pas cacher une dérive ;
- signaler franchement si le lot était mauvais ou incomplet ;
- si rien de notable n’est à signaler, l’écrire simplement.
