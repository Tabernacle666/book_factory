# faculta_boss.md

Réservoir facultatif d'idées, de phrases, d'angles, de citations, de notes d'auteur et de germes de chapitre.

Règles :
- lu dans la séquence de collaboration ;
- aucune intégration n'est automatique ;
- une idée peut être intégrée comme quote, note d'auteur, anecdote ou graine de chapitre ;
- quand une idée est intégrée, inscrire le nom et la date pour éviter les doublons ;
- certaines idées peuvent rester en attente jusqu'à former une masse critique.

Format conseillé :
- [ ] BOSS-XXX | "citation ou idée"
  - type: quote | note_auteur | anecdote | germe_chapitre
  - pourrait aller dans:
  - integre_par:
  - date_integration:
  - cible:
  - commentaire:
